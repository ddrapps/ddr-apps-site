# Tellava Frontend

Expo + React Native starter frontend for Tellava.

## Setup

1. Copy `.env.example` to `.env`
2. Ensure the API URL points to your backend:
   - `EXPO_PUBLIC_API_URL=https://tellava.onrender.com`
3. Install dependencies:
   - `npm install`
4. Start development:
   - `npx expo start`

## Purpose

This starter is intentionally small. It proves the mobile app can connect to the Render backend before the main Tellava screens are built.
