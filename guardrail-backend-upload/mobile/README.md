# Guardrail MVP

Local-first finance app that helps users avoid impulse spending with goal-based, location-triggered guardrails.

## Included
- Expo Router app scaffold
- Onboarding screen flow
- Home dashboard scaffold
- Local store and trigger engine stubs
- Product flow/spec in `docs/ui-flow-spec.md`

## Proposed stack
- Expo + Expo Router
- Zustand for app state
- expo-location for geofencing
- expo-notifications for alerts
- TypeScript

## Core promise
Your data never leaves this phone. No bank logins required.
